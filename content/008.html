Randomizing

Randomizing

The 'index' of a list is a number which describes which item in the list you're thinking about. The first item in the list has index 1, the second has index 2, and so on.

To choose a question randomly from our list of questions, we just need to pick a random number between 1 and 81 (the length of our lists) and ask the question at that index.
